📊 Data Jobs Market Analysis Dashboard

📌 Project Overview

The Data Jobs Market Analysis Dashboard is an end-to-end data analytics project that analyzes LinkedIn job postings to uncover hiring trends, in-demand technical skills, salary insights, work arrangements, and geographical hiring patterns.

The project demonstrates the complete data analytics workflow—from data exploration and preprocessing in Python to building an interactive Tableau dashboard for business intelligence.

🎯 Project Objectives

Analyze hiring trends in the data job market.
Identify the most in-demand technical skills.
Study salary distributions across different data roles.
Explore hiring locations.
Analyze work type distribution.
Build an interactive Tableau dashboard for decision-making.
📂 Dataset

Dataset: LinkedIn Job Postings (2023–2024)

The original dataset contains over 123,000 job postings across multiple industries.

For this project, the data was filtered to 1,717 data-related job postings using roles such as:

Data Analyst
Data Scientist
Data Engineer
Business Analyst

Open the notebook inside the notebooks folder.
Run the notebook to reproduce the analysis.
Open the Tableau workbook inside the dashboard folder to explore the interactive dashboard.
