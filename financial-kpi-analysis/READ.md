# 📊 Walmart Financial KPI Analysis

## 📌 Project Overview

The Walmart Financial KPI Analysis project is an end-to-end data analytics project that analyzes Walmart's weekly sales performance using Python and Tableau. The project focuses on data cleaning, exploratory data analysis (EDA), KPI calculation, and dashboard development to generate actionable business insights.

---

## 🎯 Objectives

- Analyze Walmart weekly sales data.
- Clean and preprocess the dataset.
- Calculate key financial KPIs.
- Identify top-performing stores.
- Analyze monthly and yearly sales trends.
- Compare holiday and non-holiday sales.
- Build an interactive Tableau dashboard.
- Generate business insights for decision-making.

## 🔍 Data Analysis Process

### 1. Data Collection

- Imported Walmart sales dataset.

### 2. Data Cleaning

- Checked missing values.
- Removed duplicate records.
- Converted Date column into datetime format.
- Created Year, Month, and Month_Name columns.

### 3. Exploratory Data Analysis (EDA)

- Sales summary
- Store-wise sales analysis
- Monthly sales analysis
- Holiday sales comparison
- Correlation analysis
- Distribution analysis

### 4. Dashboard Development

Designed an interactive Tableau dashboard to visualize financial KPIs and sales performance.

---

## 💡 Business Insights

- Top-performing stores generated significantly higher revenue than others.
- Sales showed seasonal trends across different months.
- Holiday weeks influenced customer purchasing behavior.
- Sales varied across different years.
- Fuel price, CPI, and unemployment showed relatively weak relationships with weekly sales.
- Interactive dashboards help managers monitor KPIs more effectively.
